Manage subscribers from your site with ease

== Description ==

DSubscribers is a simple subscribers system that collects E-mails from your website.

<a href="http://wp.dinamiko.com/demos/dsubscribers/" target="_blank">DSubscribers Demo</a>

= Features =

* Use [dsubscribers] to show the subscribe form in your Pages and Posts.
* Use [dsubscribers action="unsubscribe"] to show the unsubscribe form.
* Edit or Delete subscribers
* Search subscribers
* Export subscribers as .csv 
* Send success E-mail to the subscriber
* v1.1 Use DSubscribers Widget

== Installation ==

1. Upload the folder `dsubscribers` to the directory `/wp-content/plugins/` 
2. Activate the plugin via the 'Plugins' menu in WordPress
